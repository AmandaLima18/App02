DICAS DE PREPARAÇÃO DO AMBIENTE:

 • Crie "Áreas de trabalho virtuais" para facilitar a gestão do PC;
	
	• [Ctrl] + [WIN] + [D]  --> Cria uma nova área de trabalho virtual;
	• [Ctrl] + [WIN] + [>]  --> Acessa a próxima área de trabalho virtual;
	• [Ctrl] + [WIN] + [<]  --> Acessa a área de trabalho virtual anterior;
	• Ícone ao lado do Cortana (Visão de Tarefas) --> Gerencia áreas de trabalho.
	
 • Sugestão de organização das áreas de trabalho:
 
	• MONITOR 1 --> 2 x "Node.js command prompt" abertos;
		
		• No primeiro, rode o app --> ionic serve;
		• Deixe o segundo reservado para comandos;
		
	• MONITOR 2 --> Aplicativo aberto no Google Chrome, em modo mobile;
		
		• Ative o modo desenvoledor, teclandl [F12];
	
	• MONITOR 3 --> Visual Studio Code com diretório do aplicativo aberto;
	
	• MONITOR 4 --> "GitHub Desktop" ou console do Git (Git Bash);
	
		• Aproveite e crie um novo "branch" para esta aula;
		
	• MONITOR 5 --> "Notepad++" com pasta do aplicativo do instrutor aberta;
	
		• Caminho da pasta do instrutor: \\10.144.111.34\luferat\mobile\app01
		  * Este caminho pode mudar em outras aulas. Veja com o instrutor.
